This project calculates:
- Power (kW)
- Energy consumption (kWh)
- Electricity charge per hour and per day

## Features
- Plain PHP (No framework)
- Bootstrap 4 UI
- Hourly & daily calculation
- Based on Voltage, Current, and Rate (sen/kWh)

## Formula Used
Power (W) = Voltage × Current  
Power (kW) = Power / 1000  
Energy (kWh) = Power (kW) × Hour  
Total (RM) = Energy × (Rate / 100)
